Class JS.XMLHttpRequest

Package:JS
Defined In:XMLHttpRequest.js
Class:XMLHttpRequest
Extends:Observable
Author:jinghai.xiao@gmail.com
跨浏览器、事件驱动的XMLHTTPRequest对象，此对象完全兼容传统XMLHTTPRequest对象，在遵循 http://www.w3.org/TR/XMLHttpRequest/标准的前提下有所扩展。
